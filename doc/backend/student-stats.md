# 学生学习统计数据接口

## 接口概述

为个人信息档案界面提供学生的入驻日期、作业均分等统计数据。

---

## 接口信息

| 项目         | 说明                           |
| ------------ | ------------------------------ |
| **接口路径** | `/edu/v1/user/stats`           |
| **请求方式** | `GET`                          |
| **需要鉴权** | 是（需要携带 Token）           |
| **接口描述** | 获取当前登录学生的学习统计数据 |

---

## 请求参数

无需请求参数，通过 Token 识别当前用户。

---

## 响应参数

| 字段名             | 类型             | 必填   | 说明                                                               |
| ------------------ | ---------------- | ------ | ------------------------------------------------------------------ |
| code               | number           | 是     | 状态码，200 表示成功                                               |
| msg                | string           | 是     | 响应消息                                                           |
| data               | object           | 是     | 响应数据                                                           |
| data.joinDate      | string           | **是** | 入驻日期（用户注册/创建时间），格式：`YYYY-MM-DD`，如 `2024-01-15` |
| data.avgScore      | number \| string | **是** | 作业均分，可以是数字（如`85.5`）或字符串（如 `"暂无数据"`）        |
| data.totalHours    | number           | 否     | 累计学时（小时），可选字段                                         |
| data.totalProgress | number           | 否     | 总体学习进度（百分比，0-100），可选字段                            |

---

## 响应示例

### 成功响应

````json
{
  "code": 200,
  "msg": "success",
  "data": {
    "joinDate": "2024-01-15",
    "avgScore": 85.5,
    "totalHours": 120,
    "totalProgress": 75
  }
}

```text
### 暂无作业成绩时

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "joinDate": "2024-01-15",
    "avgScore": "暂无"
  }
}

```text
### 失败响应

```json
{
  "code": 401,
  "msg": "未登录或登录已过期",
  "data": null
}

```text

## 字段说明

### joinDate（入驻日期）

- **来源**：用户表的 `create_time` 字段
- **格式**：`YYYY-MM-DD`（只需要日期部分，不需要时间）
- **示例**：`"2024-01-15"`### avgScore（作业均分）

- **来源**：学生提交的所有作业的平均得分
- **计算逻辑**：
    - 统计该学生所有已批改作业的得分
    - 计算平均值，保留 1-2 位小数
    - 如果没有已批改的作业，返回`"暂无"`或`0`
- **示例**：`85.5`或`"暂无"`

### totalHours（累计学时）- 可选

- **来源**：学生在所有课程中累计的学习时长
- **单位**：小时
- **说明**：此字段为可选，如果后端返回，前端会优先使用此值覆盖前端计算的值

### totalProgress（总体进度）- 可选

- **来源**：学生在所有课程中的平均学习进度
- **范围**：0-100（百分比）
- **说明**：此字段为可选，如果后端返回，前端会优先使用此值覆盖前端计算的值

## 数据库参考

入驻日期可从用户表获取，示例 SQL：

```sql
SELECT DATE_FORMAT(create_time, '%Y-%m-%d') as joinDate
FROM user
WHERE id = ?

```text
作业均分计算示例 SQL：

```sql
SELECT ROUND(AVG(score), 1) as avgScore
FROM homework_submission
WHERE user_id = ? AND status = 'graded'

```---
````
